# Airline Passenger Satisfaction Analysis

## Overview
This project analyzes airline passenger reviews and satisfaction trends using SQL, Python, and Power BI.

The goal was to transform raw airline data into actionable insights through data cleaning, analysis, and interactive visualization.

---

## Tools & Technologies
- SQL
- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- DAX
- Data Modeling

---

## Key Insights
- Average passenger rating: 7.38/10
- Business Class achieved the highest entertainment satisfaction
- Seat comfort ranked as the strongest service category
- Ratings showed a gradual decline toward year-end
